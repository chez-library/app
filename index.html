About Chez Library – SJR BW

Serving the community since 2021, dedicated to education, culture, and lifelong learning for every resident. The library is free for all residents. 95% of our collection is for children of all age groups.

📍 Club House, 3rd Floor, SJR BW

BW Library – Standard Operating Procedure

🔑 Key Management
  • Given only to registered volunteers after ID verification.
  • Residents must not request or pressure Security for key access outside the volunteer system.
🤝 Volunteers
  • Only volunteers handle book issue & return.
  • Residents must not exchange or return books via Security.
⏰ Timings
  • 6:30 PM – 7:30 PM
  • Opening will be announced in the Library WhatsApp Group.
  • There may be days the library remains closed due to personal or professional preoccupations of the volunteer.
📖 Borrowing Rules
  • Maximum 1 book per person.
  • Maximum 2 books per flat at any time.
  • Borrowing period: 10 days.
  • Return or reissue only during library hours.
🎁 Book Donations
  • Accepted: Children's story books only.
  • Regional language books not accepted.
  • Hand over directly to the volunteer (not to Security).
📋 Membership
  • Free for all SJR BW residents.
  • Join via QR Code or Chez Library @BW WhatsApp Group.
⚖️ Accountability
  • Lost or damaged books must be replaced with same or equivalent copy.
🔍 You can check if you have any pending or overdue books anytime —
🤛 Library Etiquette
  • Mobile phones and electronic devices are not encouraged inside the library.
  • Please maintain a quiet and distraction-free reading environment for all.

✉ EmailJS configuration

Configure EmailJS to send automatic emails when books are issued and reminders on day 9. Sign up free at emailjs.com

Setup steps:
1. Create a free EmailJS account
2. Add an Email Service (Gmail, Outlook, etc.) and note the Service ID
3. Create an Email Template for issue confirmation and note the Template ID
4. Copy your Public Key from Account > API Keys
5. Paste all four values below and click Save
Template variables available:
{{member_name}}   {{book_title}}   {{due_date}}   {{issued_by}}   {{library_name}}   {{days_overdue}}
✓ One-time setup for all volunteers:
After saving here, open the HTML file in a text editor, find the EMAILJS_CONFIG block near the top of the <script> section, and paste the same values there. Save the file — every volunteer who opens it will have email working automatically, no setup needed on their end.

⬆ Bulk upload books

Upload a CSV file or enter books manually. All new books are auto-tagged as New Arrivals for 14 days.

CSV format — 4 columns:
BookID, Book Name, Author / Publisher, ISBN-13

Notes:
• BookID — format: Letter/Number e.g. B/266
  Letter must be A, B, C, D or E only  |  Number: 1–4 digits, cannot start with 0
  Age group is auto-set from prefix: A=0–4 · B=5–7 · C=8–12 · D=13–17 · E=Adults
  Leave blank if not yet assigned — ID can be added later from the Books panel.
• Book Name — required
• Author / Publisher — required
• ISBN-13 — optional, leave blank if unknown. Enter as plain number e.g. 9780141321011
